# BTL_XLA - Nén ảnh không mất dữ liệu (RLE & Huffman)

## Giới thiệu
Đây là bài tập lớn môn **Xử lý ảnh số**.

Đề tài:  
**Xây dựng ứng dụng nén ảnh không mất dữ liệu sử dụng thuật toán RLE và Huffman**

Ứng dụng cho phép:
- Nén ảnh BMP bằng RLE
- Nén ảnh BMP bằng Huffman
- Giải nén và tái tạo ảnh gốc
- So sánh kích thước file và tỉ lệ nén
- Hiển thị kết quả trực quan qua giao diện GUI
